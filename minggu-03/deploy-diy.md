# Deploy Some Code - DIY
1. download installer [NodeJS](https://nodejs.org/en/download), lalu install NodeJS

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/b7642a9376649a3dd474087155322f33014530de/minggu-03/Picture1.png)

2. klik finish jika anda telah selesai menginstall NodeJS

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot%202023-03-27%20213536.png)

3. setelah itu ketikkan **npm init** pada CMD dan tunggu sampai prosesnya selesai

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_9.png)

4. setelah itu ketikkan **npm install express** di CMD dan tunggu hingga proses selesai

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_10.png)

5. buka [Cyclic](https://cyclic.sh/) dan login di website tersebut, lalu pilih connect to github

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_4.png)

6. selanjutnya klik **authorize cyclic-software**

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_5.png)

7. pilih menu **link my own** dan pilih repository yang terdapat di github kita

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_6.png)

8. klik connect

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_7.png)

9. Tunggu hingga proses build dan deploying berhasil

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_11.png)

10. klik menu **OpenApp** di pojok kanan atas
11. lalu akan muncul hasil dari deploy code diatas, seperti gambar dibawah ini

![gb7](https://github.com/AnggerFNS/tekn-cloud-computing/blob/master/minggu-03/Screenshot_12.png)
